This project is MIT-licensed (see [LICENSE](LICENSE)) but builds on data and code published under other open-source licenses. This file documents those upstream sources.

## UFO Sightings — Cleaned & Unified Dataset

- **Source:** [`cjc0013/Ufo_data_clustered`](https://huggingface.co/datasets/cjc0013/Ufo_data_clustered) on Hugging Face
- **What it provides:** ~327,000 records merged from multiple publicly available UFO-sighting Kaggle datasets, with cleaning, deduplication, geocoding, contextual enrichment (moon illumination, airport proximity, weather), and text-similarity clustering
- **How this project uses it:** the dataset is fetched on demand via the Hugging Face API (or downloaded as a snapshot); raw data files are gitignored and never bundled in this repo

## NUFORC (National UFO Reporting Center)

- **Source:** [nuforc.org](https://nuforc.org/) — a public archive of UFO sighting reports
- **How this project uses it:** the `download-nuforc-csv` and `parse-csv` scripts pull NUFORC's public report CSV for cross-reference. NUFORC data is gitignored.

## uDb / Larry Hatch UFO Database

- **Original author:** Larry Hatch — `*U*` UFO database, distributed for years as a binary `U.RND` file readable only on MS-DOS
- **Modernization:** [`rr0/udb`](https://github.com/RR0/uDb) by Jérôme Beau (`javarome`), an ISC-licensed Node.js reader for the original binary database
- **How this project uses it:**
  - The `app/lib/udb/` integration calls into `@rr0/udb` (or compatible modernization) to read the binary records
  - `supabase/db/init/008_seed_udb_data.sql` is a ~15 MB Postgres seed migration generated by the `parse-udb-to-supabase.ts` and `generate-udb-seed-migration.ts` scripts in this repo, derived from records emitted by `rr0/udb`
  - Larry Hatch's underlying records date from the 1990s and have circulated publicly since; this seed migration is provided so that anyone running the project locally can populate Supabase without needing the original binary
- **License of the upstream reader:** ISC (per `rr0/udb`'s `package.json`). Copies of the `rr0/udb` source are not redistributed in this repo; only the seed data derived from running it is included. Per ISC's terms, attribution to Jérôme Beau / `rr0/udb` is preserved here.
